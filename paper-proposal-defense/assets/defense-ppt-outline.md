# 开题答辩 PPT 大纲模板

## 建议结构（12-15页，10分钟汇报）

### 第1页：封面
- 论文题目
- 姓名 / 学号
- 学科 / 专业
- 导师姓名
- 日期

### 第2页：研究背景（1页）
- 宏观背景（1-2句）
- 具体问题（1-2句）
- 研究空白（1句）
- ⭐ 用图/数据说话，不要只列文字

### 第3页：文献综述（1-2页）
- 国内外研究现状（分类综述）
- 关键文献列表（5-8篇核心）
- 研究空白定位（明确指出）
- ⭐ 用表格对比不同研究的优劣

### 第4页：研究问题（1页）
- RQ1：核心问题
- RQ2 / RQ3：子问题（如有）
- ⭐ 问题要具体、可验证

### 第5页：研究设计（1页）
- 研究对象与范围
- 变量定义（自变量、因变量、控制变量）
- 理论基础/分析框架
- ⭐ 用概念框架图展示

### 第6页：方法路径（1页）
- 方法论选择理由
- 数据收集方案
- 分析方法
- 技术路线图

### 第7页：数据来源（1页）
- 数据来源说明
- 样本方案
- 数据层级（A/B/C/D）
- 预期样本量
- Plan B（备选数据来源）

### 第8页：预期贡献（1页）
- 理论贡献（1-2条）
- 实践贡献（1-2条）
- 创新点（明确列出）

### 第9页：时间计划（1页）
- 甘特图或里程碑表
- 关键节点标注
- 当前进度

### 第10页：风险与应对（1页）
| 风险 | 概率 | 影响 | 应对 |
|------|------|------|------|
| 数据获取 | | | |
| 方法实现 | | | |
| 时间进度 | | | |

### 第11页：总结（1页）
- 题目一句话复述
- 三个关键词
- 研究价值一句话

### 末页：致谢
- "请各位老师批评指正"
- 联系方式（可选）

---

## 汇报技巧

1. **时间控制**：每页约45-60秒，文献综述可略快
2. **重点突出**：研究空白和创新点是评委最关心的
3. **诚实说明**：对局限性要主动说明，不要等评委问
4. **预案准备**：每个页面准备好可能被追问的扩展内容

## 常见 PPT 问题

- ❌ 文字太多（一页不超过50字）
- ❌ 没有图表（至少3个图/表）
- ❌ 风格不统一（字体、配色、对齐）
- ❌ 缺少页码
