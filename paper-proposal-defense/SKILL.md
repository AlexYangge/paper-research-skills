# 开题答辩模拟与风险评估 Skill

## 目的

基于 SKILL 1 输出的一页式定题卡，进行深度答辩演练，包括：
1. 生成答辩常见问题预回答框架
2. 模拟答辩 Q&A
3. 评估题目风险并给出改进建议
4. 输出 48 小时行动计划

---

## 输入要求

### 必须输入
- **一页式定题卡**：SKILL 1 或用户提供的开题题目卡
- 包含：题目、研究问题、数据来源、方法路径、预期贡献

### 可选输入
- 学科与研究层次
- 预计答辩时间（如15分钟汇报 + 10分钟问答）
- 评委背景（如导师偏好方法论严谨性 / 关注实践价值）

---

## 执行流程

### Step 1：评委视角解读

从评委视角审视选题，识别可能被质疑的薄弱点：

**核心审视问题**：
1. 这个题目真的值得研究吗？（价值质疑）
2. 为什么是你/现在做？（创新性质疑）
3. 数据真的能拿到吗？（可行性质疑）
4. 方法选得对吗？（方法质疑）
5. 做完能贡献什么？（贡献质疑）
6. 万一做不出来怎么办？（风险质疑）

### Step 2：答辩常见问题预回答框架

生成以下类别的常见问题及预回答：

#### A. 研究背景与动机类
| 问题 | 预回答要点 |
|------|-----------|
| 为什么选这个题目？ | [从文献空白 + 现实意义 + 个人基础三方面回答] |
| 这个问题的理论/实践价值是什么？ | [具体说明受益对象和影响范围] |
| 国内外研究现状如何？ | [简要综述，突出本研究位置] |

#### B. 创新性类
| 问题 | 预回答要点 |
|------|-----------|
| 你的创新点是什么？ | [明确指出与已有研究的不同] |
| 和XXX的研究有什么区别？ | [具体对比，避免泛泛而谈] |
| 凭什么说这是新的？ | [引用文献证据，不是"首次"口号] |

#### C. 方法类
| 问题 | 预回答要点 |
|------|-----------|
| 为什么选这个方法而不是XXX？ | [方法对比，说明适配性] |
| 样本量够吗？ | [解释统计功效或饱和标准] |
| 这个方法有什么局限？ | [诚实说明 + 应对措施] |

#### D. 数据类
| 问题 | 预回答要点 |
|------|-----------|
| 数据从哪来？怎么获取？ | [具体来源 + 获取步骤] |
| 数据不够怎么办？ | [Plan B 替代方案] |
| 数据质量如何保证？ | [质量控制措施] |

#### E. 可行性类
| 问题 | 预回答要点 |
|------|-----------|
| 时间够吗？ | [甘特图或时间表] |
| 需要多少经费？ | [预算说明] |
| 技术上有难点吗？ | [预判难点 + 应对策略] |

#### F. 风险与 Plan B 类
| 问题 | 预回答要点 |
|------|-----------|
| 如果假设不成立怎么办？ | [替代解释路径] |
| 如果数据拿不到怎么办？ | [备选数据源] |
| 如果结果不支持预期怎么办？ | [阴性结果也有价值] |

### Step 3：模拟答辩 Q&A

以评委角色进行模拟提问（至少5轮）：

```
## 模拟答辩 Q&A

### 第1轮：[问题类别]
**评委**：[问题]
**你的回答**：[基于定题卡的回答要点]
**评委可能的追问**：[追问方向]
**应对策略**：[如何回应]

### 第2轮：[问题类别]
...
```

### Step 4：风险评估矩阵

| 风险维度 | 风险描述 | 发生概率 | 影响程度 | 缓解措施 |
|---------|---------|---------|---------|---------|
| 选题风险 | [是否过大/过小/过旧] | 高/中/低 | 高/中/低 | [具体措施] |
| 数据风险 | [数据获取不确定性] | 高/中/低 | 高/中/低 | [Plan B] |
| 方法风险 | [方法实现难度] | 高/中/低 | 高/中/低 | [备选方法] |
| 时间风险 | [工作量估计偏差] | 高/中/低 | 高/中/低 | [里程碑检查点] |
| 创新风险 | [被抢先发表的可能] | 高/中/低 | 高/中/低 | [加速时间表] |

### Step 5：输出结果

#### 5.1 答辩准备清单

```
- [ ] 准备10分钟开题PPT（12-15页）
  - 第1页：题目 + 姓名 + 学科
  - 第2-3页：研究背景与问题
  - 第4-5页：文献综述（突出空白）
  - 第6-7页：研究设计与方法
  - 第8页：数据来源与样本
  - 第9页：预期贡献
  - 第10页：时间计划
  - 第11页：风险与应对
  - 末页：谢谢 + 请老师批评指正

- [ ] 准备3分钟口头汇报稿
- [ ] 准备每个问题的1分钟回答要点
- [ ] 预演并计时
- [ ] 找同学/导师做一次模拟答辩
```

#### 5.2 48小时冲刺计划

| 时间 | 任务 | 目标 |
|------|------|------|
| 第1天上午 | 精读核心文献3篇 | 确认研究空白表述准确 |
| 第1天下午 | 完善方法细节 | 确保方法可复现 |
| 第1天晚上 | 制作开题PPT初稿 | 完成12页框架 |
| 第2天上午 | 模拟答辩（录音/录像） | 发现表达问题 |
| 第2天下午 | 修改PPT + 补充风险应对 | 完善细节 |
| 第2天晚上 | 最终预演 | 计时 + 控场 |
