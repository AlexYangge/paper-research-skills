# 答辩常见问题库

## A. 研究背景与动机类

1. **为什么选这个题目？**
   - 考察：是否有清晰的问题意识
   - 回答模板：文献空白 → 现实需求 → 个人基础

2. **这个问题真的有研究价值吗？**
   - 考察：对领域重要性是否有认识
   - 回答模板：理论价值 + 实践价值 + 受益对象

3. **国内外研究现状如何？你是怎么做文献综述的？**
   - 考察：文献工作量
   - 回答模板：检索策略 + 核心发现 + 研究空白

4. **你的研究问题是怎么来的？**
   - 考察：问题形成过程
   - 回答模板：从宽泛兴趣 → 文献阅读 → 发现空白 → 聚焦

---

## B. 创新性类

5. **你的创新点是什么？**
   - 考察：是否有明确的新意
   - 回答模板：与已有研究对比（对象/方法/情境/数据）

6. 和 [某篇已知名论文] 的研究有什么区别？
   - 考察：文献熟悉度
   - 回答模板：具体对比表

7. 凭什么说这是新的？之前没有人做过吗？
   - 考察：文献检索是否充分
   - 回答模板：检索范围 + 对比分析

8. 你的研究是增量性的还是突破性的？
   - 考察：对贡献的定位
   - 回答模板：诚实定位 + 说明增量贡献的价值

---

## C. 方法类

9. **为什么选这个方法而不是 XXX？**
   - 考察：方法选择的合理性
   - 回答模板：方法对比 + 适配性分析

10. **你的样本量是怎么确定的？够吗？**
    - 考察：统计学基础
    - 回答模板：功效分析 / 饱和标准 / 类似研究参考

11. **这个方法的局限是什么？你会如何处理？**
    - 考察：方法论意识
    - 回答模板：承认局限 + 应对措施

12. **你的变量定义是否合理？如何保证构念效度？**
    - 考察：测量学基础
    - 回答模板：操作化定义 + 信效度检验

13. **如果因果识别有困难，你怎么回应内性质疑？**
    - 考察：因果推断意识
    - 回答模板：工具变量 / DID / RDD 等策略

---

## D. 数据类

14. **数据从哪来？怎么获取？**
    - 考察：数据可得性
    - 回答模板：来源 + 获取步骤 + 时间安排

15. **数据是二手数据，能否满足你的研究需要？**
    - 考察：数据适配性
    - 回答模板：二手数据的优势 + 局限性补偿

16. **你的样本是否有代表性？**
    - 考察：外部效度
    - 回答模板：抽样方案 + 样本特征 + 局限说明

17. **数据获取涉及伦理问题吗？如何处理？**
    - 考察：研究伦理
    - 回答模板：伦理审查 / 知情同意 / 匿名化

---

## E. 可行性类

18. **时间够吗？你的时间计划是什么？**
    - 考察：项目管理能力
    - 回答模板：甘特图 + 里程碑

19. **如果数据拿不到怎么办？**
    - 考察：风险意识
    - 回答模板：Plan B 数据源

20. **技术上有难点吗？你打算怎么解决？**
    - 考察：技术准备
    - 回答模板：预实验 + 技术路线

---

## F. 贡献与价值类

21. **你的研究预期贡献是什么？**
    - 考察：贡献定位
    - 回答模板：理论贡献 + 实践贡献分开说

22. 如果结果不支持你的假设，怎么办？
    - 考察：学术成熟度
    - 回答模板：阴性结果也有价值 + 替代解释

23. **你的研究有什么局限性？**
    - 考察：批判性思维
    - 回答模板：诚实承认 + 对未来研究的意义

---

## G. 通用压力问题

24. **你觉得自己这个研究最大的风险是什么？**
    - 回答模板：诚实指出最关键风险 + 已准备应对方案

25. **如果重来一次，你会改变什么？**
    - 回答模板：基于当前认知的改进方向

26. **你读过 [某经典文献] 吗？它和你的研究有什么关系？**
    - 回答模板：如果读过，详细讨论；如果没读过，诚实说明 + 回去补读
